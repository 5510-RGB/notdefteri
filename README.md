# Not Defteri Uygulaması

Basit ve kullanışlı bir metin düzenleme uygulaması.

## Özellikler

- **Metin Düzenleme**: Zengin metin düzenleme özellikleri
- **Dosya İşlemleri**:
  - Yeni dosya oluşturma
  - Mevcut dosyayı açma
  - Dosyayı kaydetme
  - Farklı kaydet
- **Metin Biçimlendirme**:
  - Yazı tipi boyutu ve rengi değiştirme
  - Kalın, italik, altı çizili yazı stilleri
  - Metin hizalama seçenekleri
- **Görünüm**:
  - Yakınlaştırma/Uzaklaştırma
  - Varsayılan boyuta dönme

## Kısayollar

- `Ctrl+N`: Yeni dosya
- `Ctrl+O`: Dosya aç
- `Ctrl+S`: Kaydet
- `Ctrl+Shift+S`: Farklı kaydet
- `Ctrl++`: Yakınlaştır
- `Ctrl+-`: Uzaklaştır
- `Ctrl+0`: Varsayılan boyut
- `Ctrl+Z`: Geri al
- `Ctrl+Y`: Yinele
- `Ctrl+X`: Kes
- `Ctrl+C`: Kopyala
- `Ctrl+V`: Yapıştır

- .NET 8.0 veya üzeri
